# unix环境高级编程笔记

## 一、io

### 标准io

 1. 文件流 FILE

    ​	已打开文件的标志,是一个结构体

 2. 基本io操作

    1. fopen(3)
       - 打开方式 r / r+ / w / w+ / a / a+
       - 返回值 FILE

    2. 读写	
       - 字节 fgetc(3) / fputc(3) getc(3) / putc(3)  getchar(3) / putchar(3) 
       - 行	fgets(3) / fputs(3) puts(3)	
       - 二进制	fread(3) / fwrite(3) 写的时候读到多少字节就写入多少字节
       - 格式化	fprintf(3) snprintf(3)

    3. 关闭
       - fclose(3)
    4. 缓冲
       - 作用：合并系统调用，减少程序开销，提高效率
       - 类型:
         - 行缓存	stdin stdout
         - 全缓存  默认4k
         - 无缓存  stderr
       - 强制刷新
         - fflush(NULL) 刷新所有打开的输出流
    5. 定位
       - fseek(3) 设置文件偏移量
         - SEEK_SET 文件开头
         - SEEK_CUR 当前位置
         - SEEK_END 文件末尾
       - ftell(3) 获取文件偏移量
       - rewind(3) 将文件偏移量设置到开头
    6. 临时文件
       - tmpfile(3) 当文件关闭的时候就删除了
